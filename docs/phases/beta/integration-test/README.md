# β版 integration-test

このディレクトリは、β版 implementation 完了後に実施する結合テストの計画・観点・結果を保存する場所です。

役割:
- staging 実接続での結合テスト計画
- 主要導線のテストケース
- 実施結果と不具合メモ

切り分け:
- implementation: 2026-03-15 までの実装・自動テスト・実装手順
- integration-test: β投入前の結合テスト
- release-readiness: リリース当日のチェックリストと最終スモーク

想定ファイル:
- [test_plan.md](test_plan.md)
- [test_cases.md](test_cases.md)
- [test_results.md](test_results.md)

現在:
- 結合テスト計画は [test_plan.md](test_plan.md) に記載
- 詳細手順は [test_cases.md](test_cases.md) に記載
- 実施結果は [test_results.md](test_results.md) に記録
